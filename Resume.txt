/////////////////////////////////// 3. Cara Kerja Program dan bytecode
- how to running/execute code : python "folder names"
- python adalah bahasa pemrograman interpreted, berbeda dengan compiled programming language
- # = comment in python, """ = multiline comment
- kita bisa compile python kedalam bytecode, tujuannya agar code lebih cepat di eksekusi
- cara compile code kedalam bytecode yaitu ketik "python -m py_compile "nama file" maka sistem akan lgsg membuat folder "__pycache__"
- masuk kedalama folder __pycache__ lalu ketik "python [nama file]" untuk running code

/////////////////////////////////// 4. Mengenal Variabel
- variabel adalah wadah untuk menyimpan data yang berubah-ubah nilainya
- pada python suatu data tidak perlu deklarasi atau dituliskan tipenya (integer, boolean)
- penamaan variabel pada python tidak boleh ada spasi, ada beberapa tipe penamaan suatu variabel, fungsi diantarnya pascal case, snack case, camal case
- b = a, adalah assignment indirect

/////////////////////////////////// 4. Tipe data
- tipe data angka(integer, float), string (kumpulan karakter), boolean yaitu tipe data biner 0/1 atau (True/False)
- tipe data khusus : bilangan kompleks (10 + 5i)
- format penulisan data kompleks : data_complex = complex(5,6) => 5 adalah bilangan real dan 6 adalah bilangan imaginer
- tipe data bahasa C : harus library -> from ctypes import c_double
- cara install jupyter : 
  1. install python terlebih dahulu
  2. masuk cmd ketik "pip install jupyter"
  3. setelah selesai ketik "pip install notebook"

/////////////////////////////////// 5. Casting Tipe Data
- casting adalah merubah dari satu tipe data ke tipe data lain / konversi
- data_integer= 99;
  data_float = float(data_integer);
  data_str =  str(data_integer);
- tipe data string dengan kumpulan huruf tidak bisa di casting ke dalam tipe data number (int, float) menyebabkan error
